# birthday gift

## additional setup
add file ```.env``` with variable ```VUE_APP_PARAM_NAME```, ```VUE_APP_FULL_NAME```, and ```VUE_APP_NICK_NAME``` and run project with adding query param ```name```

example
```
http://localhost:8080?name={VUE_APP_PARAM_NAME}
```
